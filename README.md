# Agon Agent 🚀

Agon Agent is an AI-powered learning platform designed to help students study smarter through personalized tutoring, adaptive quizzes, progress tracking, and intelligent study planning.

Built with React, TypeScript, Vite, Tailwind CSS, Framer Motion, and Supabase.

---

## ✨ Features

### 🧠 AI-Powered Tutoring
- Personalized learning assistance
- Instant explanations for complex topics
- Adaptive responses based on learning style

### 🎯 Adaptive Quizzes
- Dynamic difficulty adjustment
- Performance-based question generation
- Instant feedback and evaluation

### 📚 Smart Study Plans
- AI-generated study schedules
- Goal-oriented learning paths
- Time management optimization

### 💬 Natural Conversations
- Interactive AI tutor interface
- Context-aware discussions
- Real-time learning support

### 📈 Progress Tracking
- Learning analytics
- Performance monitoring
- Achievement milestones

### 🏆 Gamified Learning
- Points and rewards system
- Achievement badges
- Motivational learning experience

---

## 🛠️ Tech Stack

### Frontend
- React 19

- Vite
- Tailwind CSS

### Backend / Services
- Supabase
